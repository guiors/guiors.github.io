Project information

  • Category: Image Recognition
  • Project date: Dec. 2021

Details

I developed this school project with two students : Charlotte Ying and Axel Lopes. This model has two steps. First, the image detection part with an SSD-Mobilenet model fine-tuned with images that we had labeled. The second part is about text recognition with an OCR approach. One could compare our results with a more recent approach : an OCR-Free model called DONUT. More information on this article.
